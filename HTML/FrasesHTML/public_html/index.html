Frases HTML

EM: Se utiliza para dar -significado- a una parte del texto.

STRONG: Se utiliza para -Resaltar- una parte

Q: Se utiliza para agregar -comillas-

CODE: Se utiliza para poner el texto como si fuera código de algún -lenguaje de programación-

ABBR: Se utiliza para poner -Abreviaciones-

ACRONYM: Se utiliza para -acrónimos-

DFN: Se utiliza para poner -definiciones-

KBD: Significa texto introducido por el usuario: -el usuario-

SAMP: Se utiliza para mostrar un ejemplo de una -Salida de un programa-

VAR: Se utiliza para el texto de una -variable-
